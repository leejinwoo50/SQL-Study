# 📊 SQL-Study
SQL 문법 익히기 및 코딩 테스트 대비 저장소입니다.

## 🚀 개요
- **목적**: SQL 기초 문법 마스터 및 데이터베이스 설계 능력 함양
- **플랫폼**: [Programmers](https://school.programmers.co.kr/learn/challenges?tab=sql_high_score_kit), [SolveSQL](https://solvesql.com/)
- **언어/환경**: MySQL, VS Code, Git

## 📂 폴더 구조
- `Programmers/`: 프로그래머스 SQL 고득점 Kit 문제 풀이
  - `Lv1/`, `Lv2/` ... : 난이도별 구분
- `SolveSQL/`: 실무 데이터셋 활용 문제 풀이

## ✅ 학습 원칙
1. 문제를 풀고 난 뒤 반드시 **VS Code**에 기록한다.
2. 파일 상단에 문제 설명과 핵심 요구사항을 주석으로 남긴다.
3. 의미 있는 단위로 **Git Commit**을 남겨 히스토리를 관리한다.
